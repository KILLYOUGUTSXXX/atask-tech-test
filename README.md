# Clone Project
```bash
git clone https://github.com/KILLYOUGUTSXXX/atask-tech-test
```
---
## Installation

####  1. Install package
```bash
  npm install
```

####  2. Build package
```bash
  npm run build
```

####  3. Run app
```bash
  npm run start
```

####  4. Open application
- [localhost:3000/github](http://localhost:3000/github)


---
## Run Testing

####  1. Switch Branch
```bash
  git pull checkout -B SIT
```

####  2. re-Pull Repository
```bash
  git pull origin SIT
```

####  3. install the addition packages
```bash
  npm install
```

####  4. run the test
```bash
  npm run test
```
